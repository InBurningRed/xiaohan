## 十、技术方案

### 10.1 引擎/框架选型

| 方案 | 优势 | 劣势 | 适合场景 |
|------|------|------|----------|
| **Godot** | 轻量、开源、2D/3D 兼顾、GDScript 易上手 | 生态较小 | 2D 为主、中小体量 |
| **Unity** | 生态庞大、教程丰富、跨平台强 | 较重、授权费 | 2D/3D 通用 |
| **Unreal** | 图形顶级、蓝图友好 | 门槛高、包体大 | 3D 大作 |
| **Phaser** | Web 原生、JS/TS、轻量 | 仅 2D、无编辑器 | Web 小游戏 |
| **Three.js / Babylon.js** | Web 3D 灵活 | 无游戏编辑器 | Web 3D 项目 |
| **React + Canvas** | 可嵌入 Web 应用 | 需自行实现循环 | 轻量休闲 |
